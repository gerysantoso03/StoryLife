Discover your very deepest secret and most open stories. Go publish them

## Instalation

Set your own MongoDB URI && Google OAuth API Credentials

#### 1. Install Dependencies

```
npm install
```

#### 2. Run in Development mode

```
npm run dev
```

#### 3. Run in Production mode

```
npm start
```
